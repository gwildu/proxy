# proxy

a very simple reverse proxy tool

# Install

```
git clone https://github.com/gwildu/proxy.git
cd proxy
npm install
```

# Setup

adjust line 5 in index.js

(see redbird documantation for other configurations: https://github.com/OptimalBits/redbird)

# Run
`npm start`
